# NetScope

> A modular network traffic analysis platform built using Java.

## Overview

NetScope is a Java-based network traffic analysis platform that processes PCAP files, extracts network information, tracks communication flows, applies configurable rules, and generates analytical reports.

The project focuses on clean architecture, modular design, and enterprise software engineering practices.

## Tech Stack

- Java 21
- Maven
- Pcap4J
- SLF4J + Logback
- JUnit 5
- MySQL (Planned)
- Spring Boot (Planned)
